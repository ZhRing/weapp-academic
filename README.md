# 微信小程序－教务系统

### 说明：

实现教务系统查询学分功能，特色：
- 实现可折叠列表

### 数据接口:

- https://api.getweapp.com/vendor/academic/search

### 目录结构：

- utils — 存放项目数据接口等公共文件
- mock — 存放项目本地模拟数据文件
- pages — 存放项目页面文件
- image — 存放项目图片文件

### 开发环境：

微信web开发者工具 v0.11.112301
